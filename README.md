## 🚀 Portfólio em React JS

Este é o meu novo portfólio, desenvolvido com **React JS**, onde apresento meus principais projetos de forma moderna, dinâmica e organizada.

Durante a última semana, estive focado em construir uma aplicação que não apenas exibisse meus trabalhos, mas também proporcionasse uma boa experiência de navegação, com interface intuitiva e design responsivo.

### ✨ Sobre o projeto

O portfólio reúne alguns dos projetos mais relevantes que desenvolvi, cada um com:

* 🔗 Link para o repositório no GitHub
* 🌐 Link para o deploy (quando disponível)
* 📝 Breve descrição do projeto
* 🛠️ Tecnologias utilizadas

Além disso, o site foi estruturado utilizando boas práticas de desenvolvimento com React, incluindo componentização, organização de pastas e navegação entre páginas.

### 🧠 O que foi aplicado

* React JS com foco em componentes reutilizáveis
* React Router para navegação
* Estilização com CSS moderno
* Organização de dados via arquivos locais (simulando consumo de API)
* Boas práticas de código e estrutura de projeto

### 🎯 Objetivo

O principal objetivo deste portfólio é centralizar meus projetos e demonstrar minha evolução como desenvolvedor, especialmente no desenvolvimento front-end com React.

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.
